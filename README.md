# Password Strength Checker 
### DecodeLabs Industrial Training Kit | Batch 2026
### Cyber Security - Project 1

---

##  Project Overview
A Python program that evaluates password strength
based on security criteria and rates it as 
Weak, Medium, or Strong.

---

##  Goal
Build a program that checks whether a password 
is weak, medium, or strong using string handling 
and conditional logic.

---

## Features
- Checks password length (minimum 8 characters)
- Checks for uppercase letters
- Checks for numbers (0-9)
- Checks for special symbols (!@#$%^&*)
- Gives a score out of 4
- Provides improvement tips

---

## 🛠️ Tools Used
- Python 3.13
- VS Code

---

## ▶️ How to Run
```bash
python password_checker.py
```

---

##  How Scoring Works
| Score | Strength |
|-------|----------|
| 0-1   | WEAK     |
| 2-3   | MEDIUM   |
| 4     | STRONG   |
